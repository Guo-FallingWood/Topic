## Topic 在线论坛

### 平台介绍

Topic是一个简单的在线论坛，拥有论坛基本功能

支持数据采集，可以在后台页面操作爬虫

附带少量匹配规则，可以自定义匹配规则

效果预览：[topicbbs.top](http://topicbbs.top)

### 使用技术框架

后端

* Spring
* Spring MVC
* Mybatis
* WebMagic
* JUnit
* Hibernate Validation

前端

* JQuery
* Bootstrap

### 运行要求

- JDK 1.8
- MySql 5.6
- Tomcat 6
- Maven

## Topic 在线论坛

### 平台介绍

Topic是一个简单的在线论坛，现在仅拥有论坛基本的发帖和回复功能

Topic主要分前台用户界面和后台管理员界面两个部分

效果预览：[topicbbs.top](http://topicbbs.top)

### 使用技术框架

后端

* Spring
* Spring MVC
* Mybatis
* JUnit
* Hibernate Validation

前端

* JQuery
* Bootstrap

### 运行要求

- JDK 1.8
- MySql 5.6
- Tomcat 6
- Maven
